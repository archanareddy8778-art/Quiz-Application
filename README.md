QUIZ APPLICATION

A Quiz Application is developed using Java and Core Java concepts to conduct multiple-choice quizzes with secure user authentication, lifeline options, score calculation, and result generation. The application uses Object-Oriented Programming (OOPs) principles to provide a modular, reusable, and maintainable console-based quiz system.

## Technologies Used

* Java
* Core Java
* Object-Oriented Programming (OOPs)

## Features

* User login authentication
* Multiple-choice questions
* 50-50 lifeline
* Skip question lifeline
* Automatic score calculation
* Reward calculation
* Result generation
* Quiz flow control based on user responses

## Tools Used

* Eclipse IDE for Enterprise Java and Web Developers
* GitHub

## Project Structure

* Java is used to implement the complete quiz application.
* Object-Oriented Programming (OOPs) concepts are used to design modular and reusable classes.
* Authentication module is used for secure user login.
* Question management module is used to display and validate quiz questions.
* Scoring module is used to calculate marks and rewards.
* Result module is used to generate and display the final quiz results.
* Methods and classes are used to improve code reusability, maintainability, and application flow.
